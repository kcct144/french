## 前言

本系列只是凯凯学习语法的过程中做的精读笔记，里面尽可能详细地讲解了每一个句子，方便大家理解，不必到网上查询各种资料。

- 电脑端效果最佳，点击左侧导航栏跳转；
- 手机端点击左上角的 `Menu` 菜单打开导航栏，右上角的 `此页内容` 查看本章目录。

由于凯凯是英专生，英语和法语在词汇和语法上又有许多共通之处，所以笔记里广泛地使用了英语例词和例句，均已 `example` 的样式呈现。

::: tip 食用指南

1. 先学好拼读，本系列不教单词拼读
2. 边理解，边熟读
3. 电脑端右边有打印按钮，可以打印为 pdf 文档

:::
